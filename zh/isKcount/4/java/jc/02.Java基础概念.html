    # Java基础概念

    # Java基础语法

    一个 Java 程序可以认为是一系列对象的集合，而这些对象通过调用彼此的方法来协同工作。下面简要介绍下类、对象、方法和实例变量的概念。

    • 对象：对象是类的一个实例，有状态和行为。例如，一条狗是一个对象，它的状态有：颜色、名字、品种；行为有：摇尾巴、叫、吃等。
    • 类：类是一个模板，它描述一类对象的行为和状态。
    • 方法：方法就是行为，一个类可以有很多方法。逻辑运算、数据修改以及所有动作都是在方法中完成的。
    • 实例变量：每个对象都有独特的实例变量，对象的状态由这些实例变量的值决定。

    # 🎉程序解析

    下面看一个简单的 Java 程序，它将输出字符串 Hello World

    public class HelloWord{
    // 叫做main方法、表示程序的入口
    public static void main(String[] args){
        /* 叫做输出语句（打印语句）
        会把小括号里面的内容进行输出打印 */
        System.out.println("hello word")
    }
}

    编写 Java 程序时，应注意以下几点：

    • 大小写敏感：Java 是大小写敏感的，这就意味着标识符 Hello 与 hello 是不同的。
    • 类名：对于所有的类来说，类名的首字母应该大写。如果类名由若干单词组成，那么每个单词的首字母应该大写，例如 MyFirstJavaClass 。
    • 方法名：所有的方法名都应该以小写字母开头。如果方法名含有若干单词，则后面的每个单词首字母大写。
    • 源文件名：源文件名必须和类名相同。当保存文件的时候，你应该使用类名作为文件名保存（切记 Java 是大小写敏感的），文件名的后缀为 .java。（如果文件名和类名不相同则会导致编译错误）。
    • 主方法入口：所有的 Java 程序由 public static void main(String[] args) 方法开始执行。

    # 🎉变量

    Java 中主要有如下几种类型的变量

    • 局部变量
    • 类变量（静态变量）
    • 成员变量（非静态变量）

    1.变量的定义格式：

    数据类型 变量名 = 数据值

    public class VarnumDemo1 {
    // 主入口
    public static void main(String[] args){
        // 定义变量
        // 数据类型 变量名 = 数据值;
        // 数据类型： 限定了变量能存储数据的类型
        // int（整数） double（小数）
        // 作用: 方便以后使用
        // 数据值: 真正存在变量中的数据
        // 等号: 赋值。把右边的数据赋值给左边的变量
        int a = 10;
        System.out.println(a);
    }
}

    以上代码返回的结果是：

    10

    # 🎉计算机存储规则

    在计算机中，任意数据都是以二进制的形式存储的，进制又分有二进制、八进制、十进制、十六进制。

    99 》 0110 0011

    “a” 》 97

    汉 》 ``

    常见的进制：

    1. 二进制： 由0和1组成，代码中以0b开头
    2. 十进制： 由0-9组成，前面不加任何前缀
    3. 八进制： 由0-7组成，代码中以0开头
    4. 十六进制：由0-9还有a-f组成，代码中以0x开头

    任意进制转十进制

    公式： 系数 * 基数的权次幂 相加

    • 系数：就是每一位上的数
    • 基数：当前进制数
    • 权： 从右往左，依次为0 1 2 3 4 5

    二进制转十进制

    可以使用8421快速转换法

    11111111
    1286432168421

    十进制转其他进制

    可以使用除基取余法

    数据类型的四类八种：

    整数和小数的大小关系：

    这里分别说明两个问题两个注意事项，整数和小数取值范围大小关系：

    1. double > float > long > int > short > byte
    2. 其中byte的取值范围是-128---127
    3. long类型变量： 需要加入L标识 （大小写都可以）
    4. float类型变量： 需要加入L标识 （大小写都可以）

    关于8种数据类型的分类输出：

    public class HelloWord{
    // 主入口
    public static void main(String[] args) {
        // byte
        byte b = 10;
        System.out.println(b); //10 

        // short
        short s = 20;
        System.out.println(s); //20 

        // int
        int i = 30;
        System.out.println(i); //30 

        // long
        /* 如果要定义long类型的变量
        在数据值的后面需要加一个L作为后缀
        L可以大写的，也可以是小写的
        建议：使用大写
        */

        long n = 9999099999L;
        System.out.println(n);
        
        // folat
        // 注意点；定义float类型需要加一个F后缀 
        float f = 10.1F;
        System.out.println(f);

        // double 
        double d = 10.02;
        System.out.println(d);

        // char 
        char c = '中';
        System.out.println(c);

        // boolean
        boolean o = true;
        System.out.println(o); 

    }
}

    以上代码返回的结果是：

    # 🎉标识符

    什么是标识符？

    标识符： 就是给类，方法，变量等起的名字。

    标识符命名规则 --- 硬性要求

    1. 由数字、字母、下划线（_）和美元符（$）组成
    2. 不能以数字开头
    3. 不能是关键字
    4. 区分大小写

    标识符命名规则 --- 软性要求

    1. 小驼峰命名法： 方法、变
    2. 大驼峰命名法： 类名

    # 🎉键盘录入

    键盘录入介绍：

    Java帮我们写好了一个类叫做Scanner，这个类就可以接收键盘输入的数字。类似Python的input。

    使用方法:

    • 步骤一： 导入包 --- Scanner

      import java.util.Scanner;
// 导入包的动作必须出现在类定义上方

    • 步骤二： 创建对象 --- 表示我们要开始使用这个类了

      Scanner sc = new Scanner(System in);
// 上面这个格式里面，只有sc是变量名，可以变，其他的都不允许变

    • 步骤三： 接收数据 --- 真正开始干活了

      int i = sc.netxtInt();
// 上面这个格式里面、只有i是变量名 可以变，其他都不允许变

    键盘录入一个整数：

    import java.util.Scanner;

public class ScannerDemo {
    public static void main(String[] args) {
        // 创建对象，表示我现在准备要用Scanner这个类
        Scanner sc = new Scanner(System.in);

        // 接收数据
        // 变量i记录键盘录入的数据
        System.out.println("请输入整数:");
        int i = sc.nextInt();
        System.out.println(i);
    }
}

    以上代码返回的结果：

    请输入整数:
123
123
