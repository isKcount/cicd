    # Java字符串

    # 字符串详解

    # 🪵String概述

    java.lang.String类代表字符串，Java程序中的所有字符串文字（例如："abc"）都为此类的对象。

    字符串广泛应用 在 Java 编程中，在 Java 中字符串属于对象，Java 提供了 String 类来创建和操作字符串。

    String类的特点详解：

    String其实常被称为不可变字符串类型，它的对象在创建后不能被更改。

    字符串学习内容：

    • String
    • StringBuilder
    • StringJonier
    • StringBuffer
    • Pattern
    • Matcher

    # 🪵创建字符串

    创建字符串最简单的方式如下:

    String str = "Runoob";

    在代码中遇到字符串常量时，这里的值是 " Runoob "，编译器会使用该值创建一个 String 对象。

    和其它对象一样，可以使用关键字和构造方法来创建 String 对象。

    用构造函数创建字符串：

    String str2=new String("Runoob");

    String 创建的字符串存储在公共池中，而 new 创建的字符串对象在堆上：

    String s1 = "Runoob";              // String 直接创建
String s2 = "Runoob";              // String 直接创建
String s3 = s1;                    // 相同引用
String s4 = new String("Runoob");   // String 对象创建
String s5 = new String("Runoob");   // String 对象创建

    String 类有 11 种构造方法，这些方法提供不同的参数来初始化字符串，比如提供一个字符数组参数:

    package test10;

public class StringDemo5 {
    public static void main(String[] args) {
        char[] helloArry = {'r','u','n','o','o','b'};
        String helloString = new String(helloArry);
        System.out.println(helloString);
    }
}

    以上代码返回的结果：

    runoob

    字符串的使用

    • str.charAt(Index) 用于索引
    • str.substring(beginIndex,endIndex) 用于截取
    • str.replace(oldChar, newChar) 用于替换

    # 🪵字符串截取

    基本语法：

    String substring(int beginindex，int endindex) 截取

    注意点： 包头不包尾，包左不包右边，只有返回值才是截取的小串

    public class StringDemo2 {
    public static void main(String[] args) {
        // 获取一个手机号
        String phoneNumber = "18933200910";

        // 截取手机号的前三位
        String start = phoneNumber.substring(0, 3);

        // 截取手机号的后四位
        String end = phoneNumber.substring(7);

        // 拼接
        String result = start + "****" + end;
        System.out.println(result);
    }
}

    以上代码返回的结果：

    189****0910

    # 🪵字符串的比较

    推荐使用String类提供的“equals”比较：只关心内容一样即可

    方法名说明
    public boolean equals (Object anObject)将此字符串与指定对象进行比较。只关心字符内容是否一致！
    public boolean equalsIgnoreCase (String anotherString)将此字符串与指定对象进行比较，忽略大小写比较字符串。只关心字符内容是否一致！

    假设已经知道正确的用户密码，请用程序实现模拟用户登录。总共给三次机会，登录之后会给出相应的提示。

    package test10;

import javax.swing.*;
import java.util.Scanner;

public class StringDemo2 {
    public static void main(String[] args) {
        // 已经知道正确的用户密码，请用程序实现模拟用户登录。总共给三次机会，登录之后会给出相应的提示

        // 定义两个变量
        String username = "admin";
        String password = "adminpassword";

        int count = 0;
        // 模拟用户键盘录入
        while (true){
            Scanner sc = new Scanner(System.in);
            System.out.println("=============");
            System.out.println("请输入用户名：");
            String user = sc.next();
            System.out.println("=============");
            System.out.println("请输入密码： ");
            String pass = sc.next();

            // 判断两个字符串比较是否正确
            boolean result1 = username.equals(user);
            boolean result2 = password.equals(pass);
            if (result1 && result2){
                System.out.println("登录成功");
                break;
            }else {
                count++;
            }
            if (count == 3){
                System.out.println("你的用户密码输入错误已经超过三次，已锁定");
                break;
            }
        }
    }
}

    以上代码返回的结果：

请输入用户名：
admin
=============
请输入密码： 
adminpassword
登录成功

    # 🪵字符串的替换

    字符串的替换，可以使用String.replace ，调用方法添加旧的字符串以及需要替换的新字符串

    示例：

    如下为一个敏感词替换的小程序，替换敏感词汇为***。

    public class StringDemo3 {
    public static void main(String[] args) {
        String[] arr = {"TMD","SB","CNM","MLGB"};

        String talk = "你玩的真好，CNM,TMD以后不要再玩了";
        for (int i = 0; i < arr.length; i++) {
            talk = talk.replace(arr[i],"***");
        }
        System.out.println(talk);
    }
}

    以上代码返回的结果：

    你玩的真好，***,***以后不要再玩了

    # 🪵遍历字符串

    • public char charAt(int index): 根据索引返回字符
    • public int length(): 返回此字符串的长度
    • 数组的长度： 数组名.length
    • 字符串的长度： 字符串对象.length()

    需求：键盘录入一个字符串，使用程序实现控制台遍历该字符串，统计小写字母，大写字母以及数字出现的次数。

    package test10;

import java.util.Scanner;

public class StringDemo4 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入一个字符串");
        String str = sc.next();

        int bigCount = 0;
        int smallCount = 0;
        int numberCount = 0;
        // 统计

        for (int i = 0; i < str.length(); i++) {
            char c = str.charAt(i);
            System.out.println(c);
            if (c >= 'a' && c <= 'z'){
               smallCount++;
            }else if (c >= 'A'  && c <= 'Z') {
                bigCount++;
            }else if(c >= '0' && c <= '9'){
                numberCount++;
            }
        }
        System.out.println("小写字母一共有" + smallCount);
        System.out.println("大写字母一共有" + bigCount);
        System.out.println("数字一共有" + numberCount);

    }
}

    # 🪵拼接字符串

    定义一个方法，把int数组中的数据按照指定的格式拼接成一个字符串返回，调用该方法，并在控制台输出结果。

    例如：

    数组为 int[] arr = {1,2,3};

    执行方法后的输出结果为： [1,2,3]

    package test10;

public class StringDemo6 {
    public static void main(String[] args) {
        // 遍历一个数组，拼接成一个字符串

        // 创建一个数组
        int[] arr = {1, 2, 3, 4, 5, 6, 7};
        String str = arrToString(arr);
        System.out.println(str);
    }

    public static String arrToString(int[] arr){
        if(arr == null){
            return "";
        }
        if (arr.length == 0){
            return "[]";
        }
        String result = "[";
        for (int i = 0; i < arr.length; i++) {
            if (i == arr.length -1){
                result = result + arr[i];
            }else {
                result = result + arr[i] + ",";
            }
        }
        result = result + "]";
        return result;
    }
}

    以上代码返回的结果：

    [1,2,3,4,5,6,7]

    # 🪵字符串反转

    定义一个方法，实现字符串反转，键盘录入一个字符串，调用该方法后，在控制台输出结果。

    package test10;

public class StringDemo7 {

    public static void main(String[] args) {
        String result = reverser("abc");
        System.out.println(result);

    }

    public static String reverser(String str){
        String result = "";
        for (int i = str.length() - 1; i >= 0; i--) {
            char c = str.charAt(i);
            result = result + c;
        }
        return result;
    }
}

    以上代码返回结果：

    cba

    # 🪵金额转换

    package test10;

import java.util.Scanner;

public class StringDemo8 {
    public static void main(String[] args) {
        // 键盘录入一个金额
        Scanner sc = new Scanner(System.in);
        int money;
        while (true){
            System.out.println("请录入一个金额");
            money = sc.nextInt();
            if (money >= 0 && money <= 9999999){
                break;
            }else {
                System.out.println("金额无效");
            }
        }
        String moneyStr = "";
        // 得到money里面的每一个数字
        while (true){
            // 从右往左获取数据，因为右侧是数据的个位
            int ge = money % 10;
            String capitalNumber = getCapitalNumber(ge);
            // 把转换之后的大写拼接到moneyStr中
            moneyStr = capitalNumber + moneyStr;

            // 去掉刚刚获取的数据
            money = money / 10;

            // 如果数字上的每一位全部都获取到了 那么money记录的就是0 此时循环结束
            if (money == 0){
                break;
            }
        }
        int count = 7 - moneyStr.length();
        for (int i = 0; i < count; i++) {
            moneyStr = "零" + moneyStr;
        }

        // 插入单位
        // 定于一个数组表示单位
        String result = "";
        String[] arr = {"佰","拾","万","仟","佰","拾","元"};
        for (int i = 0; i < moneyStr.length(); i++) {
            char c = moneyStr.charAt(i);
            result = result + c + arr[i];
        }
        System.out.println(result);
    }

    public static String getCapitalNumber(int number){
        // 定义数组，让数字跟大写的中文产生一个对应的关系
        String[] arr = {"零","壹","贰","叁","肆","伍","陆","柒","捌","玖"};
        return arr[number];
    }
}

    以上代码返回的结果：

    请录入一个金额
16545
零佰零拾壹万陆仟伍佰肆拾伍元

    # 🪵StringBuilder概述

    StringBuilder可以看成是一个容器，创建之后里面的内容是可变的

    # 🪵StringBuilder使用

    作用： 提高字符串的效率

    StringBuilder的构造方法

    方法名说明
    public StringBuilder()创建一个空白可变的字符串对象，不含有任何内容
    public StringBuilder(String str)根据字符串内容，来创建可变的字符串对象
    public StringBuilder append(任意类型)添加数据，并且返回对象本身
    public StringBuilder reverse()反转容器中的内容

    构造一个StringBuilder的方法，对他依次添加，反转，转换为字符串。

    public class StringDemo4 {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("abc");

        sb.append(1);
        sb.append("abc");
        System.out.println(sb);
        sb.reverse();
        // 打印一下他
        System.out.println(sb);

        String str = sb.toString();
        System.out.println(str);
    }
}

    以上代码返回的结果：

    abc1abc
cba1cba
cba1cba

    # 🪵链式编程

    当我们在调用一个方法的时候，不需要用变量接受它的结果，可以继续调用其它方法

    一般两种情况会用到StringBuilder

    1. 字符串的拼接
    2. 字符串的反转

    输入一个字符串，通过添加，反转，转换成字符串，判断是否为对称字符串。

    import java.util.Scanner;

public class StringDemo6 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入一个字符串");
        String str = sc.next();

        // 获取字符串 
       	// 如下的方法就是链式编程
        String newstr = new StringBuilder().append(str).reverse().toString();
        System.out.println(newstr);

        // 比较
        if (str.equals(newstr)){
            System.out.println("当前是一个对称的字符串");
        }else {
            System.out.println("当前不是一个对称的字符串");
        }
    }
}

    以上代码返回的结果：

    请输入一个字符串
123321
123321
当前是一个对称的字符串

    对一个数组进行拼接操作，拼接成字符串的形式。

    public class StringDemo7 {
    public static void main(String[] args) {
        int[] arr = {1,2,3};
        String str = arrtoString(arr);
        System.out.println(str);

    }

    public static String arrtoString(int[] arr){
        StringBuilder sb = new StringBuilder();
        sb.append("[");

        for (int i = 0; i < arr.length; i++) {
            if (i == arr.length -1){
                sb.append(arr[i]);
            }else {
                sb.append(arr[i]).append(",");
            }
        }
        sb.append("]");
        return sb.toString();
    }
}

    以上代码返回的结果：

    [1,2,3]

    # StringJoiner

    # 🪵StringJoiner的概述

    • StringJoiner跟StringBuilder一样，也可以看做成是一个容器，创建之后里面的内容是可变的。
    • 作用：提高字符串的操作效率，而且代码编写特变简洁，但是目前市面上很少有人用。
    • JDK8出现的

    # 🪵StringJoiner的使用

    StringJoiner的构造方法

    方法名说明
    public StringJoiner(间隔符号)创建一个StringJoiner对象，指定拼接时的间隔符号
    public StringJoiner(间隔符号 开始符号 结束符号)创建一个StringJoiner对象，指定拼接时的间隔符号，开始符号，结束符号

    添加间隔符

    import java.util.StringJoiner;

public class StringDemo10 {
    public static void main(String[] args) {
        // 创建一个对象 指定中间的间隔符号
        StringJoiner sj = new StringJoiner("----");

        // 添加元素
        sj.add("aaaa").add("bbbb").add("cccc");

        System.out.println(sj);
        
    }
}

    以上代码返回的结果：

    aaaa----bbbb----cccc

    # 🪵StirngJoiner的成员方法

    方法名说明
    public StringJoiner add (添加的内容)添加数据，并返回对象本身
    public int length()返回长度(字符出现的个数)
    public String toString()返回一个字符串(该字符串就是拼接之后的结果)

    StringJoiner的方法参数说明：

    • delimiter 间隔符号
    • prefix 开始符号
    • suffix 结束符号

    拼接一个字符串。

    import java.util.StringJoiner;

public class StringJoinerDemo1 {
    public static void main(String[] args) {
        // delimiter 间隔符号
        // prefix 开始符号
        // suffix 结束符号
        StringJoiner sj = new StringJoiner(",", "[", "]");
        sj.add("aaa").add("bbb").add("ccc");

        int len = sj.length();
        System.out.println(len);


        System.out.println(sj);
        sj.toString();
    }

    以上代码返回的结果：

    13
[aaa,bbb,ccc]

    # 🪵总结

    String: 表示字符串的类，定义了很多操作字符串的方法

    StringBuilder: 一个可变的操作字符串的容器，可以高效的拼接字符串，还可以将容器里面的内容反转

    StringJoiner: JDK8出现的一个可变的操作字符串的容器，可以高效，方便的拼接字符串。

    # 🪵字符串原理

    扩展底层原理1：字符串存储的内存原理

    • 直接赋值会复用字符串常量池中
    • new出来不会复用，而是开辟一个新的空间

    扩展底层原理2： ==号比较的到底是什么？

    • 基本数据类型比较数据值
    • 引用数据类型比较地址值

    扩展底层原理3： 字符串拼接的底层原理

    • 如果没有变量参与，都是字符串直接相加，编译之后就是拼接之后的结果，会服用串池中的字符串
    • 如果有变量参与，每一行拼接的代码，都会在内存中创建新的字符串，浪费内存

    扩展底层原理4： StringBuilder提高效率原理图

    • 所有要拼接的内容都会往StringBuilder中放，不会创建很多无用的空间，节约内存。

    扩展底层源码5： StringBuilder源码分析

    • 默认创建一个长度为16的字节数组
    • 添加的内容长度小于16，直接存
    • 添加的内容大于16会扩容，（原来的容量*2 + 2）
    • 如果扩容之后还不够，以实际的长度为准
